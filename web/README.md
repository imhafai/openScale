# openScale Web PoC

This directory contains a simple proof-of-concept web interface built with [Svelte](https://svelte.dev/) and [Vite](https://vite.dev/).
It lets you record weight entries locally in your browser as a first step toward a web version of openScale.

## Development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
